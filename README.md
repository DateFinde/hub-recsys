# hub-recsys

## 简介
通往机器学习算法工程师的进阶之路是崎岖险阻的，为了系统化的记录自己在推荐系统方向的学习和实践经验，特意用这个项目进行记录，主要涵盖「推荐系统基本概念」、「经典推荐算法」、「深度推荐算法」三大部分，本仓库主要对算法进行实现。各个算法模型的原理和结构，优势和不足，实践和应用等方面记录在推荐系统专栏。【更新填坑ing ~】
